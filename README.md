# AI Learning App

AI Learning App is a full-stack web application that helps users learn more effectively from PDF documents using AI-powered tools.
Users can upload a PDF and generate quizzes, notes, summaries, and interact with an AI chat assistant to gain deeper insights into specific topics.

The application also provides a dashboard where users can track their learning progress, recent activities, and test their knowledge through AI-generated quizzes.
All AI-related features are powered by the **Gemini AI API**.
